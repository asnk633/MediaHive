# Awesome DESIGN.md

DESIGN.md is a plain-text design system document that AI agents read to generate consistent UI.
It's just a markdown file. No Figma exports, no JSON schemas, no special tooling. Drop it into your project root and any AI coding agent or Google Stitch instantly understands how your UI should look. Markdown is the format LLMs read best, so there's nothing to parse or configure.

| File | Who reads it | What it defines |
|------|-------------|-----------------|
| `AGENTS.md` | Coding agents | How to build the project |
| `DESIGN.md` | Design agents | How the project should look and feel |

## Aesthetic Direction Guidelines

When creating or matching a DESIGN.md template for your UI:

### AI & LLM Platforms
- **Claude**: Warm terracotta accent, clean editorial layout, sophisticated card hierarchy.
- **Cohere**: Vibrant gradients, data-rich dashboard aesthetic.
- **ElevenLabs**: Dark cinematic UI, audio-waveform aesthetics, glowing accents.
- **Mistral AI**: Open-weight LLM provider. French-engineered minimalism, purple-toned.
- **Ollama**: Terminal-first, monochrome simplicity.
- **Runway**: Cinematic dark heroes, paper-white reading bands, single proprietary sans, and pure black pill CTAs.
- **Together AI**: Technical, blueprint-style design.

### Developer Tools & IDEs
- **Cursor**: Sleek dark interface, gradient accents.
- **Expo**: Dark theme, tight letter-spacing, code-centric.
- **Raycast**: Productivity launcher. Sleek dark chrome, vibrant gradient accents.
- **Superhuman**: Premium dark UI, keyboard-first, purple glow.
- **Vercel**: Frontend deployment platform. Black and white precision, Geist font.
- **Warp**: Modern terminal. Dark IDE-like interface, block-based command UI.

### Design Principles (Google Stitch)
1. **Never use placeholders**: Avoid generic mockups. Use rich visuals.
2. **Harmonious Palette**: Prefer tailwind-like custom color ranges (e.g. HSL tailored) rather than generic HTML colors.
3. **Typography**: Google Fonts like Inter, Roboto, Outfit, and custom typography to feel premium.
4. **Vibrant glassmorphism**: Layered semi-transparent backdrops with clean blur and border highlights.
