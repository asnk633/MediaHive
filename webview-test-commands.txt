# WebView Emulator Test Commands

## Run all WebView emulator tests
npx playwright test e2e/playwright/webview-emulator.spec.ts

## Run tests for specific WebView type
npx playwright test e2e/playwright/webview-emulator.spec.ts --project=chromium

## Run with verbose output
npx playwright test e2e/playwright/webview-emulator.spec.ts --reporter=verbose

## Run in headed mode for debugging
npx playwright test e2e/playwright/webview-emulator.spec.ts --headed

## Run on Windows
npm run test:e2e:win

## View test results
npx playwright show-report

## Run with custom reporter
npx playwright test e2e/playwright/webview-emulator.spec.ts --reporter=line

## Run with trace enabled for debugging
npx playwright test e2e/playwright/webview-emulator.spec.ts --trace on

## Run specific test by line number
npx playwright test e2e/playwright/webview-emulator.spec.ts:15

## Run tests in debug mode
npx playwright test e2e/playwright/webview-emulator.spec.ts --debug
